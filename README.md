# MIDI Chord Trainer

A real-time chord training app built with React and the Web MIDI API.  
Connect your MIDI keyboard and improve your chord recognition, accuracy, and speed.

---

## Features

- Random chord generation (major, minor, 7th, diminished, augmented, etc.)
- Adjustable timer for each round
- Real-time MIDI input detection
- Instant feedback ("Correct" / "Incorrect")
- Note matching logic using pitch class comparison
- Live MIDI device connection detection

---

## How It Works

1. The app generates a random chord
2. You play the chord on your MIDI keyboard
3. The app compares your input to the expected notes
4. You receive instant feedback before time runs out

---

## Tech Stack

- **React (TypeScript)**
- **Vite**
- **Web MIDI API**
